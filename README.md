# plupload-java-springboot
plupload for java and use file's md5 to keep unique。 前端用plupload,后端使用springboot搭建，计算了文件的md5保证了文件的唯一行，使用定时合并已经上传的文件，文件上传中途即使电脑死机也可以开机重新从上次上传断点出继续上传。
